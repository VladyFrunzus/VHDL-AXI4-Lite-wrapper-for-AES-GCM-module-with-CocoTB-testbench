# VHDL-AXI4-Lite-wrapper-for-AES-GCM-module-with-CocoTB-testbench
This repository contains a AXI4-Lite wrapper for an AES-GCM module with native signals, which uses internal registers to store cryptodata. Additionally, this project also contains a testbench for CocoTB, complete with the Makefile and Test Runner.
